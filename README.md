Alarm Clock Suite — 7 языков, многофункциональный будильник
Коллекция из семи реализаций будильника с расширенным функционалом:

Установка времени срабатывания (часы, минуты, секунды)

Выбор звукового сигнала (встроенный писк / пользовательский файл / синтез)

Кнопки Установить, Отключить, Отложить (Snooze) на 5 минут

Повтор: ежедневно, по будням, один раз

Отображение текущего времени и времени будильника

Визуальная/звуковая индикация при срабатывании

Сохранение настроек в файл / localStorage

Языки и интерфейсы
Язык	Интерфейс	Особенности
Python	Tkinter GUI	Звук через pygame или winsound, повтор, snooze, сохранение в JSON
JavaScript	Веб (HTML/CSS)	Notification API, выбор аудиофайла, Web Audio, localStorage
Go	CLI + отображение	Асинхронная проверка времени, системный beep, поддержка конфигурации
Rust	CLI + цветной вывод	chrono для времени, звук через rodio, планирование в отдельном потоке
Java	Swing GUI	javax.sound.midi или Toolkit.beep, многопоточность, трей иконка
C#	Windows Forms	System.Media.SoundPlayer, таймер, повтор, иконка в трее
PHP	Веб (сервер + cron)	Хранение в сессии, имитация звука через JS, синхронизация с серверным временем
